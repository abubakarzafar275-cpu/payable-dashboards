📖 View Sheets Setup Guide (Steps 1 & 2)
Step 1 — Set up your Google Sheet

Your Google Sheet must have 4 tabs with these exact names:
PO_Report  Daily_Log  NonPO_Queue  COA_Accounts

Step 2 — Deploy the Apps Script
  1. Open your Google Sheet → click Extensions → Apps Script
  2. Delete all existing code in the editor
  3. Open the file google_apps_script.gs (in your app folder) and copy all its contents
  4. Paste it into the Apps Script editor → click Save (💾)
  5. Click Deploy → New Deployment
  6. Set Execute as: Me  |  Who has access: Anyone
  7. Click Deploy → Authorize → Copy the Web App URL
